##### How to run #######################

Simply double click on them and MATLAB runtime (around ~3GB) should get installed.
You won't need a MATLAB installation to run these apps. 

####### Special instructions for PAC_plots app ##############

1. You should create a nX1 matrix for both EEG and EGG data and saved as .mat file. 
   'n' is the size of the signal and it should match for both of the data.
2. The EEG data should be saved as 'eeg_section' and EGG data as 'egg_section'.
3. The data should be filtered in your prerequisite ranges.
3. There are sample data files provided. Please refer to them.
